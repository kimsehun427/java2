# 자바의 예외 처리, try-catch-finally문
예외 처리 : 발생한 예외에 대해 개발자가 작성한 프로그램 코드에서 대응하는 것
try-catch-finally문 사용. finally블록은 생략 가능.
try{
    예외가 발생한 가능성이 있는 실행문(rty 블록)
}
catch (처리할 예외 타입 선언) {
    예외 처리문(catch블록)
}
finally {
    예외 발생 여부와 상관없이 무조건 실행되는 문장(finally 블록) //생략 가능
}
